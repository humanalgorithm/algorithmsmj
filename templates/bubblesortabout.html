Bubble sort is one of the simpler sorting algorithms to implement and is an O(n^2) sorting algorithm. The reason it is called Bubble Sort is because the larger elements tend to "bubble up" to the end of the array. If the array is already sorted then Bubble sort can operate in a best case running time of O(n). On the other hand the worst case scenario is if the array is in reverse sorted order in which the running time is still O(n^2).

The reason that Bubble Sort can potentially operate in O(n) time is that it checks to see if the array is already sorted when it runs. If it can make it to the end of the array and it sees that it is already sorted then it can exit. Use Bubble Sort »

def bubbleSort(listIn):
    temp = 0
    #initialize outer loop counter
    i = 0
    #initialize inner loop counter
    j=0
    #set the number of passes to make to the length of the array
    numPasses = len(listIn)
    #set boolean holder whether array is sorted or not
    sorted = False

    #loop through outer loop the length of the array times
    while i < numPasses:
        #if inner loop went through and each element was sorted then break out of outer loop
        if sorted == True:
            break
        #assume sorted true when going into inner loop
        sorted = True
        #initialize inner loop counter
        j=0

        #last i elements will always be sorted so do inner loop length of list-i-1
        while j < len(listIn)-1-i:
            #if previous element is greater than next element then do a swap
            if listIn[j] > listIn[j+1]:
                temp = listIn[j+1]
                listIn[j+1] = listIn[j]
                listIn[j] = temp
                #note that array wasn't sorted
                sorted = False
            j+=1
        i+=1
    return listIn
