How to use this page

  • Select the size of the array that you would like to have created. The website will generate a random array for you with numbers from 1 to 10000

  • Click on the sorting function that you would like to execute on that array

  • Note the time to execute that it takes to execute the algorithm

  • Click on the "Unsort array" button if you would like to unsort the array

  • Try other sorting functions and see how the execution time is different for different sorting algorithms

  • The max size of the array is 1,000,000 and the timeout time is 30 seconds
